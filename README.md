# 甲骨文媒体库监控大屏

纯原生HTML + Python Flask风格HTTP服务实现的Emby媒体库监控大屏，完美布局三栏结构。

## 🎯 布局特点（严格按照要求）

| 📍 位置 | 模块 | 尺寸说明 |
|--------|------|---------|
| **左栏 280px** | 🌍 3D地球 | ✅ 固定 **260×260px**，绝对不压缩 |
| | 👥 谁在看 Emby | ✅ 地球正下方，最大高度 240px，列表可滚动，不超长 |
| | 📊 媒体库总览 | ✅ 占剩余空间，自动滚动 |
| **中栏（自适应）** | 🎬 Emby 最近入库 | ✅ 在上，海报墙最小110px，自动填充 |
| | 📝 日志中心 | ✅ 在下，支持tab切换，可滚动 |
| **右栏 280px** | 🩺 STRM 健康度 → ⏰ 定时任务 → ☁️ 云盘统计 → 💻 系统信息 → 🔐 安全证书 | ✅ 每个面板平分高度，都不是长条，完整显示 |

## ✨ 功能

- 🌍 3D旋转地球，在线用户点位标注
- 👥 实时显示谁在观看Emby
- 🎬 最近入库电影海报墙展示
- 📝 多tab日志中心（Emby/rclone/STRM）
- 📊 完整的媒体库和系统监控
- 🌓 支持深色/浅色主题切换
- 纯原生开发，不需要npm，不需要框架

## 🚀 部署

```bash
# 克隆到服务器
git clone https://github.com/你的用户名/media-dashboard.git
cd media-dashboard

# 修改配置
# 编辑 serve_media.py 修改 EMBY_KEY 和 EMBY_HOST

# 启动服务
nohup python3 serve_media.py > serve.log 2>&1 &

# 访问
http://你的服务器IP:8771/
```

## 📋 依赖

- Python 3.7+
- Emby/Jellyfin 媒体服务器
- 可选：rclone 云盘挂载，STRM自动同步
